O elemento table (tabela )

o elemento table (tabela)representa dados tabulares ou seja,informaçãoes que são representados em linhas e colunas

Exemplo

Tabela simple com cabeçalho

Primeiro nome> sobrenome
Priscila Sena
Luis Paiva

Elementos da tabela

  • tabela - tabela;
  • (tabela header) - cabeçalho da coluna;
  • (tabela row) - linha da tabela da coluna
  • (tabela data ) - valor/ conteúdo da célula
