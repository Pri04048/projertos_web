Lista não ordenadas

A tag ul(unordered list - lista ordenada ) representa uma lista de intens som ordem rigida,por exemplo ingredientes em uma receita.

A tag li (list itens- intens da lista) representa um item que faz parte de uma lista

Exemplo de lista

  • primeiro item
  • segundo item
  • Terceiro item
    • Subitem

Lista ordenadas

A tag ol (ordered lista - lista ordenada) representa uma de itens numerados sequencialmente.

Exemplo de lista ordenada

  1. Primeiro item
  2. Segundo item

Atributos

reversed (ordem reversa)
Exibe os intens da lista ordenada em ordem reserva

  • Item mais importante
  • Item menos importante
  • Item mais importante
  • Item menos importante

    • Star (Início)
    específico o valor Inicial para numeração da lista.

    1. Item 5
    2. Item 6

    type(tipo) /
    indica o tipo de numeração:
    a - Indica letras mínusculas:
    A- indica çetras maiúsculas:
    i-indica algarismo romanos minúsculos:
    I - indica algarismos romanos maiúsculos:
