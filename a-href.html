Tag a href define um hiperlink(conexão), que é ultilizado para vincular uma página a outra.

o atributo mais importante desta tag é o href que define a página de destino que será exibida ápos o clique do mouse no link

Por padrão, os links aparecerão da seguinte dorma em todos os navegadores

um link não visitado é sublinado de azul

um link visitado é subliado de roxo

um link ativo é sunbliado de vermelho

Exemplo

Link para arquivo h1-h6

Clique aquipara visitar a página sobre Marcadores H1-H6

Link para o site do SENAI Suzano

Clique aqui pata visitar a páginado SENAI Suzano
